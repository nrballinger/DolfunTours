# DolFun Tours Website

Vercel-ready React/Vite website with content managed in `src/design.json`.

## Edit the design/content
Open `src/design.json` and change:
- phone number
- booking URL
- prices
- tour descriptions
- photos
- reviews
- FAQ

## Run locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Create a new GitHub repo.
2. Upload these files.
3. Import the repo into Vercel.
4. Framework preset: Vite.
5. Build command: `npm run build`.
6. Output directory: `dist`.

## Replace photos
The current design uses Unsplash placeholder photos. Replace them with real DolFun Tours photos inside `src/design.json` for the best results.
