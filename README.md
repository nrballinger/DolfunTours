# DolFun Tours Static Website

This version has no npm dependencies. It is plain HTML, CSS, JavaScript, and `design.json`.

## Deploy to Vercel

1. Upload this folder to GitHub.
2. Import the GitHub repo in Vercel.
3. In Vercel project settings:
   - Framework Preset: Other
   - Build Command: leave blank
   - Output Directory: leave blank or `.`
   - Install Command: leave blank
4. Deploy.

## Edit content

Edit `design.json` to change business info, tour pricing, reviews, FAQs, and hero wording.

## Add real photos

Replace the gradient placeholders by adding images to `/assets` and updating the CSS `.hero` and `.photo-card` backgrounds.
